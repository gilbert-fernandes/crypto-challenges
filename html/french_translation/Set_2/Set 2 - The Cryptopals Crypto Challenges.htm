Epreuves cryptographiques : Jeu 2

Ce jeu est le premier d'un ensemble consacré au chiffrement par blocs. Il s'agit de la cryptographie de tous les jours. Celle qui est implémentée dans la plupart des logiciels web contenant de la cryptographie.

Ce jeu est relativement facile. Les personnes qui terminent le jeu 1 n'ont aucune difficulté à finir le jeu 2.

Trois des exercices au sein de ce jeu sont extrêmement importants pour briser de la cryptographie utilisée actuellement; le premier permet de déchiffrer les messages chiffrés dans le mode par défaut de l'AES, et les deux autres permettent de réécrire des messages chiffrés au-travers des modes les plus populaires de l'AES.

  1. Implémenter un remplissage PKCS#7
  2. Implémenter le mode CBC
  3. Oracle de détection ECB/CBC
  4. Déchiffrement ECB octet-par-octet (simple)
  5. Couper-coller ECB
  6. Déchiffrement ECB octet par octet (difficile)
  7. Validation de remplissage PKCS#7
  8. Attaque de CBC par inversion de bits
