Validation de remplissage PKCS#7

Ecrivez une fonction qui prend un texte clair, et détermine si son remplissage (padding) est valide, puis retire le remplissage s'il est valide.

La chaîne :

"ICE ICE BABY\x04\x04\x04\x04"

... possède un remplissage valide, et produit pour résultat "ICE ICE BABY".

La chaîne :

"ICE ICE BABY\x05\x05\x05\x05"

... ne possède pas un remplissage valide, pas plus que la chaîne suivante :

"ICE ICE BABY\x01\x02\x03\x04"

Si vous utilisez un langage disposant d'exceptions, comme le Python ou Ruby, écrivez votre fonction afin qu'elle génère une exception quand un remplissage invalide est détecté.
