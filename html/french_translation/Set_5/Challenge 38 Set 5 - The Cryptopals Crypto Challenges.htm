Attaque par dictionnaire hors-ligne sur un SRP simplifié

S 
x = SHA256(sel|mot de passe)
    v = g**x % n
C->S 
I, A = g**a % n
S->C 
sel, B = g**b % n, u = nombre aléatoire 128-bit
C 
x = SHA256(sel|mot de passe)
    S = B**(a + ux) % n
    K = SHA256(S)
S 
S = (A * v ** u)**b % n
    K = SHA256(S)
C->S
Envoyer HMAC-SHA256(K, sel)
S->C
Envoyer "OK" si le HMAC-SHA256(K, sel) est valide

Note : dans ce protocole, le paramètre "B" du serveur ne dépend pas du mot de passe (c'est juste une clef publique Diffie Hellman).

Vérifiez que le protocole fonctionne quand on lui donne un mot de passe valide.

Maintenant, faites fonctionner le protocole comme un attaquant MITM Faites-vous passer pour le serveur, et utilisez des valeurs arbitaires pour b, B, U et le sel.

Cassez le mot de passe depuis le A du HMAC-SHA256(K, sel).
