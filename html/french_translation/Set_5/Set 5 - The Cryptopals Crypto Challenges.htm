Epreuves cryptographiques : Jeu 5

Cet ensemble est le premier consacré à la cryptographie reposant sur la théorie des nombres et couvre également l'authentification des messages.

Cet ensemble d'épreuves est sensiblement plus difficile que le précédent. Les concepts sont nouveaux, les attaques ne ressemblent à rien de ce que vous avez utilisé jusqu'à présent et... plus mathématiques.

D'autre part, notre attaque cryptographique préférée se trouve dans cet ensemble (vous allez bientôt la découvrir). Nous aimons beaucoup cet ensemble. N'échouez pas, vous avez presque terminé !

  1. Implémenter Diffie-Hellman
  2. Implémenter une attaque MITM à clef fixe sur Diffie-Hellman avec injection de paramètre
  3. Implémenter DH avec négociation de groupes, puis le briser via des paramètres "g" malicieux
  4. Implémenter un mot de passe distant sécurisé (SRP - Secure Remote Password)
  5. Briser SRP avec une clef nulle
  6. Attaque par dictionnaire hors-ligne sur un SRP simplifié
  7. Implémenter le RSA
  8. Implémenter une attaque RSA broadcast avec E=3
