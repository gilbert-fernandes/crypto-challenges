Inversion de bit CTR

Il existe des personnes convaincues dans ce monde que le CTR est capable de résister aux attaques par inversion de bit, attaques pour lesquelles le mode CBC est vulnérable.

Ré-implémentez l'exercice d'inversion de bit CBC pour utiliser le mode CTR au lieu de CBC. Puis, injectez dans le texte clair la chaîne "admin=true".
