Briser un MD4 préfixé MAC par extension de longueur

Attaque identique à la précédente, mais on remplace le SHA-1 par MD4. Ayant déjà mené cette attaque contre le SHA-1, la variante MD4 devrait prendre beaucoup moins de temps; probablement le temps qu'il vous faut pour mettre la main sur une implémentation de MD4.

Vous vous demandez : pourquoi s'embêter avec tout ceci ?

C'est la faute de Stripe. Dans leur second jeu CTF, l'avant-dernière épreuve consistait à briser un MAC SHA-1 H(k,m). Ce qui signifie que le code SHA-1 flotte un peu partout sur Internet. Mais du code pour MD4, pas vraiment.
