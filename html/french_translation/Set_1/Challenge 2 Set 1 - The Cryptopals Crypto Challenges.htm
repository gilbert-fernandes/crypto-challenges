XOR fixe

Ecrire une fonction qui prend deux buffers de même taille, et produit leur combinaison par XOR.

Si votre code fonctionne correctement, vous pourrez lui fournir la chaîne : 

1c0111001f010100061a024b53535009181c

... après décodage hexadécimal et XOR avec la valeur suivante : 

686974207468652062756c6c277320657965

... obtenir pour résultat : 

746865206b696420646f6e277420706c6179
