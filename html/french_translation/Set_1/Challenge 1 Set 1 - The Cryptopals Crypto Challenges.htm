Conversion hexadécimale vers base64

La chaîne : 

49276d206b696c6c696e6720796f757220627261696e206c696b65206120706f69736f6e6f7573206d757368726f6f6d

Doit produire : 

SSdtIGtpbGxpbmcgeW91ciBicmFpbiBsaWtlIGEgcG9pc29ub3VzIG11c2hyb29t

Produisez le résultat attendu. Le code créé sera utilisé au cours des exercices suivants.

Règle à suivre

Toujours opérer sur des octets bruts, jamais sur des chaînes encodées. N'utilisez les valeurs hexadécimales et base64 que pour l'affichage auprès de l'utilisateur.
