Epreuves cryptographiques : Jeu 3

Voici la série suivante d'épreuves reposant sur la cryptographie à chiffrement par bloc (y compris les parties traitant des générateurs pseudo-aléatoires).

Ce jeu est modérérement difficile. Il contient la célèbre attaque contre le mode CBC, une attaque par "clonage" d'un générateur pseudo-aléatoire populaire qui est difficile à implémenter correctement.

Vous atteignez ici le point où toutes les épreuves, à l'exception d'un exercice, sont utiles et exploitables pour briser de la cryptographie en usage dans le monde réel.

  1. Oracle CBC à remplissage
  2. Implémenter CTR, le chiffrement de flux
  3. Briser le mode CTR à nonce fixée par substitution
  4. Briser le CTR à nonce-fixée statistiquement
  5. Implémenter un générateur aléatoire MT19937 "Mersenne Twister"
  6. Briser une racine MT19937
  7. Cloner un générateur MT19937 en analysant sa sortie
  8. Créez un chiffrement de flux MT19937 puis brisez-le
