Implémenter un générateur aléatoire MT19937 "Mersenne Twister"

Vous trouverez le code pseudo-aléatoire sur Wikipedia.

Si vous écrivez votre code en Python, Ruby ou (beurk!) PHP, votre langage contient probablement déjà le MT19937 dans la fonction "rand()"; n'utilisez-pas rand(). Vous devez l'implémenter vous-même.
