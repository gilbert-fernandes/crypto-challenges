Créez un chiffrement de flux MT19937 puis brisez-le

Vous pouvez créer un chiffrement de flux facilement à partir de n'importe quel générateur pseudo aléatoire. Utilisez-le pour générer une séquence de sorties de 8-bits et considérez ces sorties comme un keystream. Puis faites un XOR de chaque octet du texte clair avec chaque octet successif du keystream.

Ecrivez une fonction qui implémente ce concept en utilisant un générateur MT19937 et une racine de 16-bits. Vérifiez que vous pouvez chiffrer et déchiffrer correctement. Ce code sera assez similaire au code utilisé pour le CTR.

Utilisez votre fonction pour chiffrer un texte clair connu (par exemple, 14 lettres "A" consécutives) préfixé par un nombre aléatoire de caractères aléatoires.

Depuis le texte chiffré, retrouvez la "clef" (la racine de 16-bits).

Utilisez la même idée pour générer un mécanisme de remise à zéro de mot de passe, reposant sur un générateur MT19937 dont la racine est l'heure actuelle.

Ecrivez une fonction qui permet de tester si un jeton "mot de passe" est le produit d'un générateur MT19937 dont la racine a été alimentée par l'heure actuelle.
