Récupération de nonce DSA depuis une nonce répétée

Récompense cryptanalytique MVP

Cette attaque (et dans un groupe à courbe elliptique) est celle qui a brisé la PS3. C'est une très, très grande attaque.

Dans ce fichier se trouve une collection de signatures DSA. (NB: chaque message est suivi par une espace)

(en utilisant les mêmes paramètres de domaine que l'exercice précédent)

Il ne devrait pas être très difficile de trouver les messages pour lequels nous avons utilisés un "k" de manière répétée. Si vous avez une paire de ces messages, vous pouvez découvrir le "k" que nous avons utilisé, avec la formule suivante : 

         (m1 - m2)
     k = --------- mod q
         (s1 - s2)

Mathématiques de première année de lycée : étudiez !

Si vous voulez démystifier tout ceci, obtenez cette équation en partant des équations DSA elles-mêmes.

Les opérations mathématiques en groupe cyclique veulent vous piéger

N'oubliez pas que toutes ces mathématiques sont modulo q; s2 peut être plus grand que s1, par exemple, mais ce n'est pas un problème si vous faites votre soustraction modulo q. Si vous êtes comme moi, vous allez probablement perdre une bonne heure à cause d'une parenthèse oubliée ou un modulo q. (et n'oubliez pas cette fonction d'inversion modulo !)
